## Packages
recharts | For data visualization charts (Revenue, ROI, etc.)
framer-motion | For smooth page transitions and micro-interactions
date-fns | For date formatting in tables and reports
clsx | For conditional class merging
tailwind-merge | For safe Tailwind class merging

## Notes
The application uses Replit Auth for authentication.
API endpoints are defined in @shared/routes.
Real-time updates are simulated via query invalidation for this version, as full WebSocket setup requires a separate server implementation not fully scoped here, but the UI is ready for it.
